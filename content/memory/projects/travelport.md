At Travelport (2022 – present), I work as Senior Frontend Engineer on platform-scale SDK and plugin workflows.

In platform work, API ergonomics and reliability matter as much as raw feature count.

A recurring tradeoff: strict contracts improve long-term stability, but can slow short-term iteration. The best outcomes came from explicit versioning and clearer partner-facing boundaries.
